Core Features
왜 WATCH2Project인가?
프로젝트 착수부터 종료까지, 공공 SI의 모든 관리 영역을 하나의 플랫폼에서. 역할 기반 접근 제어와 실시간 현황 모니터링으로 프로젝트 성공률을 극대화합니다.
📊

WBS/일정 관리

트리 구조 WBS 편성, 간트차트, 마일스톤, 선후행 관계, 영업일 기반 자동 진척률 계산. 엑셀 Import/Export와 드래그 일정 변경을 지원합니다.

📁

산출물 관리

방법론별 산출물 테일러링, 버전 이력, 다단계 승인 워크플로(PL→QA→PMO→고객), 감리 체크리스트 연동. 산출물 누락을 자동 검출합니다.

📈

공정 진척현황 S-Curve

업무별 가중 평균 진척률, 계획 대비 실적 S-Curve 차트, 기준일별 누적 진척 추이. 주차별 데이터 테이블과 업무별/담당자별 분석을 제공합니다.

💻

개발진척현황 NEW

1,500+ 프로그램 단위 개발 목록 관리, 엑셀 일괄 등록, 담당자별 완료일 등록, S-Curve 통계, 업무별/담당자별 완료율 차트. 프로젝트 캘린더 연동.

📝

주간/월간보고

자동 데이터 수집(진척/이슈/위험), 팀별 실적/계획 작성 워크플로, 작성 요청→작성중→취합→완료. DOCX 가로 문서 다운로드와 알림 연동.

⚠️

이슈/리스크 관리

7개 범주별 이슈/리스크 등록, 상태 변경 이력, 중요도/긴급성 매트릭스, 공유 대상 관리. 주간보고 및 대시보드와 자동 연동.

📅

프로젝트 캘린더

한국 공휴일/주말 표시, 마일스톤 타임라인, 이벤트 등록(감리/회의/릴리즈), 날짜 선택 시 주말/공휴일 자동 차단. PmsDatePicker 공통 컴포넌트.

📋

요구사항 관리

요구사항 ID 체계, 진행상태(수용/제외/통합/대체), 변경이력 추적, 베이스라인 관리. 엑셀 Import/Export와 개발 프로그램 매핑.

🔔

실시간 알림 NEW

헤더 벨 아이콘 알림 배지, 1분 자동 폴링, 주간보고 작성 요청/완료 알림, 대시보드 위젯. 읽음 처리 및 전체 읽음 기능.

🏢

PMO 통합 대시보드

프로젝트 현황, 진척률, 결함/이슈/위험 요약, 승인 대기, 내 태스크, 일정/마일스톤 위젯. 위젯 설정/접기/펼치기 커스터마이징.

🔍

감리 대응

공정별 필수 산출물 매핑, 감리 체크리스트(적합/부적합/조건부), 감리원 전용 뷰, 감리보고서 자동 생성. 방법론에 따른 산출물 테일러링 정의.

⚙️

시스템 관리

사용자/권한(7개 역할), 공통코드, 감사로그(전 구간 CUD 기록), 방법론/산출물 정의, 승인 프로세스 설정, 업무별 가중치, DB 백업.

Weekly Report

주간보고, 자동화된 워크플로

PMS관리자가 보고서를 생성하고 팀에 작성 요청을 발송하면, 각 팀장이 실적/계획을 작성하고, 관리자가 취합하여 최종 완료합니다. WBS 태스크가 자동으로 수집되어 작성 부담을 최소화합니다.

  • 6단계 워크플로: 생성→작성요청→작성중→취합→검토→완료
  • WBS depth 4 기준 금주/차주 태스크 자동 수집
  • 팀별 작성 현황 실시간 모니터링
  • 임시저장 / 작성완료 분리 (알림 연동)
  • 내 팀만 보기 ON/OFF 스위치
  • DOCX 가로형 문서 다운로드 (실적/계획 병렬 배치)
  • 전체 진척률 + 팀별 진척률 + 업무별 진척률 표시
  • 개발진척현황 (관리자 설정으로 ON/OFF)

Integration Benefits

1 + 1 > 2, 통합의 시너지

프로젝트 관리(WATCH2Project)와 품질 관리(WATCH2Defect)를 연동하면, 개별 운영 대비 훨씬 강력한 관리 체계를 구축할 수 있습니다.

  • WBS ↔ 테스트케이스 연동 — 태스크별 테스트 커버리지 자동 추적, 산출물 승인 시 TC 실행률 검증
  • 요구사항 ↔ RTM 통합 — 요구사항 ID로 설계→개발→테스트→결함 전 구간 추적, 누락 자동 검출
  • 진척률 ↔ 품질 지표 통합 — 주간보고에 결함 현황/테스트 실행률 자동 포함, S-Curve에 품질 메트릭 병행 표시
  • 감리 대응 원스톱 — 산출물 현황 + 테스트 결과 + 결함 조치율을 하나의 감리 보고서로 통합 생성
  • 대시보드 통합 뷰 — 프로젝트 진척, 개발 완료율, 테스트 성공률, 미해결 결함을 단일 화면에서 모니터링
  • 동일 기술 스택 — Vue 3 + Node.js + PostgreSQL, Docker Compose로 함께 배포, 사용자/인증 통합
